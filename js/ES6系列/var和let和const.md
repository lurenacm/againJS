## let
* `let` 声明的变量也会变量提升，但是不会初始化成 undefined。这是和 var 的区别。
* 声明的变量在 `{}` 是块级作用域。但对象这些引用类型值不是
* 声明后不能再次声明同一个变量，变量的值可以修改
* 经常使用在 `for` 循环中，每一个循环都会形成独立的块级作用域，互不干扰

## const 
* `const` 声明的变量是常数，声明时必须赋值
* 声明后不能再次声明同一个变量，变量的值不可以修改
